Portafolio

Periodo I

En el primer periodo, me introduje en la gestión de servidores y contenedores. Investigué Kubernetes para orquestar contenedores y realicé la instalación de Windows Server en entornos VMware y VirtualBox. Exploré Docker para el despliegue de aplicaciones, incluida la publicación de una página web en un contenedor.

Periodo II

Durante el segundo periodo, adquirí conocimientos sobre servidores web, centrándome en la configuración de Apache Tomcat y el servidor web IIS. También exploré la instalación de Ubuntu Server y publiqué una aplicación War en Apache Tomcat.

Periodo III

En el tercer periodo, realicé investigaciones sobre servidores web avanzados, configurando Apache2 y Nginx en Ubuntu Server 22.04 LTS. Además, exploré el despliegue de sitios web con Load Balancer en Microsoft Azure y AWS, profundizando en la configuración de grupos de destinos y Application Load Balancer en el contexto del cloud computing.
